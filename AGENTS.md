# Mapa Mental Interativo — Agentes e Skills OpenCode

Stack: **React 18 + Vite + @xyflow/react + Zustand + Tailwind + jsPDF**

## Agentes Recomendados para Este Projeto

| Agente | Quando Usar |
|--------|-------------|
| `/frontend-developer` | Componentes React Flow, UI interativa |
| `/performance-optimizer` | Performance do canvas com muitos nós |
| `/testing-specialist` | Testes de componentes e estado Zustand |
| `/accessibility` | WCAG para editor de mapa mental |
| `/code-reviewer` | Review de lógica de nós e arestas |

## Skills Recomendadas

| Skill | Quando Usar |
|-------|-------------|
| `frontend-design` | Novos componentes, estilos de nós |
| `performance` | Virtualização para grafos grandes |
| `testing` | Testes com Vitest + Testing Library |
| `accessibility` | Keyboard navigation no canvas |
| `javascript-typescript` | Lógica de manipulação do grafo |
| `git-workflow` | Commits e releases |

## Estrutura do Projeto

```
MAPA MENTAL/
├── src/
│   ├── components/       ← Componentes React Flow
│   ├── store/            ← Zustand state
│   └── utils/
├── index.html
├── opencode.json         ← configuração OpenCode
└── package.json
```
